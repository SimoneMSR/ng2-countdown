# ng2-countdown (Angular2 Countdown)
Angular2 contdown is complatible with latest release of angular 2.x.x.

## Usage
Follow these steps:

### 1. Update your `systemjs.config.js` file.
Add following line in map:

```js
map: {
      //...
      'ng2-countdown': 'npm:ng2-countdown'
     }
```
-and in packages:

```js
packages: {
      //...
      'ng2-youtube': {
        main: './index.js',
        defaultExtension: 'js'
      }
```
